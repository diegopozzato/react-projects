## Run Locally

**Clone the project**

```bash
  git clone https://github.com/diegopozzato/react-basic-calculator.git
```

**Go to the project directory**

```bash
  cd react-basic-calculator
```

**Install dependencies**

```bash
  npm install
```

**Start the server**

```bash
  npm start
```
